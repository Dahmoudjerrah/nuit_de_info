Comment utiliser le projet
Installation
Aucune installation nécessaire ! C'est un fichier HTML autonome.
Lancement

Téléchargez le fichier formulaire-gloire.html
Double-cliquez sur le fichier pour l'ouvrir dans votre navigateur
Ou faites un clic droit → "Ouvrir avec" → Votre navigateur préféré


 Fonctionnalités principales
1. Validation complète du formulaire

 Validation en temps réel de tous les champs
 Messages d'erreur clairs et visuels
 Animation "shake" sur les champs invalides
 Bordures rouges pour indiquer les erreurs

2. Popup de succès

 Message "Félicitations ! Vous avez gagné !"
 Design attractif avec trophée
 Animations fluides d'apparition et de disparition
 3 étoiles animées qui clignotent
 Fermeture par bouton ou clic sur le fond

3. Design responsive

 S'adapte parfaitement aux mobiles, tablettes et ordinateurs
 Bouton pleine largeur sur mobile
 Layout optimisé pour tous les écrans

 Easter Eggs (Surprises cachées)
Easter Egg  : Confettis explosifs 
Comment le trouver :

Remplissez tous les champs du formulaire correctement
Cliquez sur "Envoyer le message"
BOOM ! Des confettis multicolores explosent de partout !

Effet : Animation de confettis depuis le centre et les côtés de l'écran

Easter Egg  : Compteur de caractères dynamique 
Comment le trouver :

Cliquez dans le champ "Message"
Commencez à taper du texte
En bas à droite, vous verrez : "X / 500 caractères"

Effet bonus :

Le compteur change de couleur quand vous dépassez 400 caractères (devient orange)
Limite maximale : 500 caractères


Easter Egg  : Mode sombre secret 
Comment le trouver :

Descendez tout en bas de la page (footer)
Cherchez la petite icône de lune (très discrète)
Cliquez 3 fois rapidement sur cette icône

Effet :

Tout le site passe en mode sombre avec un dégradé bleu/noir
Les formulaires deviennent gris foncé
Le texte s'adapte automatiquement
Recliquez 3 fois pour revenir au mode clair


Easter Egg  : État de chargement réaliste 
Comment le trouver :

Remplissez le formulaire correctement
Cliquez sur "Envoyer le message"
Observez le bouton

Effet :

Le bouton affiche "Envoi en cours..." avec un spinner animé
Le bouton est désactivé pendant secondes
Simule un vrai envoi de données
Puis la popup de félicitations apparaît


Easter Egg  : Effet glow sur les champs 
Comment le trouver :

Cliquez sur n'importe quel champ du formulaire (Nom, Email, Sujet, Message)

Effet :

Une ombre lumineuse indigo apparaît autour du champ actif
Animation fluide de transition
Retour à la normale quand vous quittez le champ

Technologies utilisées

HTML5 - Structure du document
Tailwind CSS (via CDN) - Framework CSS pour le design
JavaScript Vanilla - Logique et interactivité
Font Awesome - Icônes professionnelles
Canvas Confetti - Bibliothèque pour l'effet confetti
Google Fonts (Inter) - Typographie moderne
